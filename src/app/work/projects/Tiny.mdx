## Overview

I needed a cheap way to get a reliable minecraft server and services running 25/7.
I went exploring online looking for a small computer, at first I thought buying a mini pc with mobile processor but I found them quite
expensive comparing hardware/price so I discovered a close company that sold used hardware, got a good deal on a Lenovo Tiny. I went with
this option for being durable and easy repairable, which includes a removable processor capable of upgrade in the future.
It does consume a little more power compared to a regular mobile processor but it is not noticeable at the end of the month.
Giving the philosophy of Lenovo and enterprise-grade material it's a no brainer.

## Key Features

- **Feature 1**: Self hosted Minecraft server
- **Feature 2**: DeGoogled services
- **Feature 3**: Streaming services

## Technologies Used

- **Debian**: Reliable and stable, NO Gui
- **Docker**: Keep things isolated and organized
- **Termius**: App to connect via SSH

## Challenges and Learnings

Learned how to connect via SSH and created a script to keep the public IP up to date in the domain.

## Outcome

Power efficient server, powerful enough for home use. Use of free and open source projects.
Minecraft server running 24/7 with no additional cost.
Use of n8n for automations!
